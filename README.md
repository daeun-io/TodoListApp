TodoListApp is a task management application that helps you organize your daily to-do lists.
- Create, Update, Delete Todos
- Category-Based Management: Load and manage your todo lists based on custom categories
- Kakao Login Integration
- Firebase Cloud Storage: When logged in with Kakao, your todos are saved and synced with Firebase.
- Notificaiton Alarms: Receive notificatiosn at the time you set for each todo
